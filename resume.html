Portfolio

Apps, Projects, and other items

Vexlus

Vexlus, a mashup of visualizing text as a vector nexus, is my current SaaS project. Instead simple word counts, Vexlus uses keyword and NER APIs to extract topics from any text. Then, it uses a custom scoring technique to relate the topics, creating a clustering effect. This effect creates a soft relationship within the text, allowing a user to visualize many rows of text entries in a unique way that can bring out patterns buried within.

Synchronizer

Synchronizer extends Microsoft's Sync Framework to allow local databases to go remote, and resync with the server at a later time. The app is designed to address issues with foreign key relationships within auto numbering referential databases. Integrity is maintained through Sync Framework tracking tables, row versioning, and custom triggers defined in the class.

RaspiBot

RasiBot is an autonomous robot that used a sacrificed webcam and a few ultrasonic sensors to follow colors. Although the bot has been retired, it's code lives on in the repo linked above. I used OpenCV to detect colors within the frame. Each frame gets iterated over, and each pixel is looked at to see if it matches the color to follow. After the frame is processed, it averages the found (x,y), and tells the bot which way to go.
